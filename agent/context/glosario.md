# Glosario - Observatorio de Educacion

| Termino | Definicion |
|---|---|
| ICET | Indice de Calidad Educativa Territorial. Indice compuesto 0-100 del proyecto |
| ITT | Indice de Transformacion Territorial. Indice padre al que alimenta este proyecto |
| SED | Secretaria de Educacion Municipal de Santiago de Cali |
| MEN | Ministerio de Educacion Nacional de Colombia |
| IE | Institucion Educativa. Agrupa varias sedes bajo una misma direccion |
| Sede | Unidad fisica donde se presta el servicio educativo. Identificada por codigo DANE |
| DANE | Codigo unico de identificacion de sedes educativas asignado por el DANE |
| Score | Valor normalizado de 0 a 100 para un indicador o dimension |
| ref_min | Umbral inferior de normalizacion (mejor resultado razonable o meta aspiracional) |
| ref_max | Umbral superior de normalizacion (peor resultado tolerable) |
| clamp | Funcion que acota un valor al rango [0, 100] |
| Indicador inverso | Indicador donde menor valor significa mejor resultado (desercion, repitencia) |
| Indicador positivo | Indicador donde mayor valor significa mejor resultado (matricula, cobertura) |
| Tasa de cobertura bruta | Matricula total / Poblacion en edad escolar x 100 |
| Tasa de cobertura neta | Matricula en edad adecuada / Poblacion en edad escolar x 100 |
| Tasa de desercion | Estudiantes que abandonan / Matricula inicial x 100 |
| Tasa de repitencia | Estudiantes que repiten / Matricula total x 100 |
| Tasa de aprobacion | Estudiantes que aprueban / Matricula total x 100 |
| Ratio estudiantes/docente | Total estudiantes / Total docentes en la sede |
| Ratio estudiantes/equipo | Total estudiantes / Total equipos de computo en la sede |
| Emprestito | Credito publico para inversion en infraestructura educativa |
| Jornada | Turno de operacion de la sede (manana, tarde, completa, nocturna) |
| Comuna | Division administrativa territorial del municipio de Cali |
| CRS | Sistema de referencia de coordenadas (EPSG) |
| Corte | Momento temporal al que corresponden los datos (ej: corte 2026) |
